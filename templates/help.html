Помощь

Эта программа отвечает на вопросы на основе содержания документации в папке data.

Для этого она читает файлы (поддерживаются txt, rtf, doc, ppt, xls, pdf, html и многое другое), делит их на чанки и векторизует эти чанки с помощью эмбеддинговой нейросети, формируя векторную базу данных.

Запрос пользователя также векторизуется и сравнивается с базой данных. Несколько наиболее похожих чанков передается как контекст для LLM, которая с их помощью отвечает на исходный запрос.

При первом запуске, или при изменении/ добавлении файлов в папке data во время работы программы, нажмите ОБНОВИТЬ БАЗУ ДАННЫХ.
